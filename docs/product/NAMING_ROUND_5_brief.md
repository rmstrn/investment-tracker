## Criteria (LOCKED, per `03_NAMING.md` refined 2026-04-23)

Hard criteria (non-negotiable):
- **Length: 1 word ideal, 2 words max.**
- **Memorable + meaningful** — not generic abstract, not pure descriptive. User should be able to say «called [X], and it's about [feeling/idea]».
- **Bilingual pronounceability: EN + RU primary.** English is day-1 launch language per PO 2026-04-23; RU is secondary parallel-drafted; EU languages deferred. Name must work in both — no awkward transliteration stumble, no Russian letter-cluster that mangles English rendering.
- Pronounceable without schwa-heavy English-only phonetics (no «thr-«, «-eigh», tongue-twisters).
- **NOT** generic fintech suffixes: `-ly`, `-io`, `fin-`, `wealth-`, `invest-`.
- Matches tone: Magician + Sage primary, Everyman modifier (see `02_POSITIONING.md` §Brand archetype — **note archetype updated in v2 from «Magician + Everyman»**).
- Domain `.com` / `.app` / `.ai` realistically obtainable (ideally <$10K — indirect WebFetch signal only; final verification via Namecheap/Porkbun/Cloudflare Registrar is free).
- No strong trademark conflicts in US (USPTO) + EU (EUIPO) — free search via their public endpoints.

Soft criteria (preferred):
- Semantic hook connected to memory / brain / knowledge / noticing / recall.
- Not in any of four rejected rounds (`03_NAMING.md` §Directions explicitly tried and rejected).
- Short and memorable over long and literal.
